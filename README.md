# switchtwitch

![Switch Twitch Banner](https://github.com/weiliddat/switchtwitch/blob/main/public/switchtwitch-banner.png?raw=true)

Switch Twitch channels smoothly, with auto mute/unmute, quality changes, previews, and more!

## Development

1. Clone repo
2. npm install
3. npm run dev
